Reciclagem

A reciclagem envolve várias etapas e é influenciada por diferentes fatores, como o interesse do mercado, os custos envolvidos, as leis de cada região e, acima de tudo, a colaboração das pessoas, que precisam fazer o descarte correto dos materiais. Essa participação é essencial para garantir que o material reciclado tenha boa qualidade. Isso é importante porque os produtos reciclados precisam ser competitivos em relação aos produtos novos no mercado.

Sobre o material

Como descartar o material

  1. Separação: Separe seus materiais entre recicláveis (papéis, metais, vidro e plásticos) e não recicláveis (restos de comidas no geral)
  2. Higienização: Limpe os materiais recicláveis que ainda tenham algum resto de alimento
  3. Embalagem: Use sacos de lixo adequados para os seus materiais
  4. Descarte: Leve os seus materiais para o centro de reciclagem mais próximo de você

Que destino o lixo tem após a coleta?

Após a coleta, o lixo reciclado é geralmente mandado para uma cooperativa ou para alguma central de triagem, onde os materiais são separados por tipo para serem levados para o processo de reciclagem. O lixo não reciclável como resíduos orgânicos e dejetos são mandados para aterros sanitários, que são locais feitos para minimizar o impacto ambiental. Em alguns casos o lixo orgânico pode ser usado como adubo. No caso de lixo hospitalar, geralmente eles são levados para incineradores apropriados para serem incinerados.
